### 安装dnsmasq
- apt-get install -y dnsmasq
#### /etc/dnsmasq.conf
```conf
bogus-priv
no-resolv
server=100.100.2.138
server=100.100.2.136
interface=tun0
listen-address=127.0.0.1
bind-interfaces
addn-hosts=/etc/internal_ips
```
#### /etc/internal_ips
10.8.0.1 main.int  
10.8.0.2 cj.int

10.8.0.4 s1.int
